# Sistema-mini-ERP
Esse sistema visa unificar processos que  podem ser utilizados pelas empresas do mercado. O objetivo da aplicação é pegar um sistema de estoque simulado e por meio da modularização exibir funções essenciais de gestão de estoque. Com isso, temos: dashboards dinâmicos, meio de cadastro de produto, lista dos produtos em estoque  e outras funções.
